# Zeeshan Restaurant Srikakulam

Premium static React + Tailwind website for **Zeeshan Restaurant Srikakulam - Apna Hyderabadi Food**.

## Run locally

Use the bundled/static server:

```bash
npm run dev
```

Then open:

```text
http://localhost:4173
```

## Deploy on Vercel

1. Upload this project folder or push it to GitHub.
2. In Vercel, create a new project and select this folder/repository.
3. Framework preset: **Other**.
4. Build command: `npm run build`
5. Output directory: leave empty / default.
6. Deploy.

The project includes `vercel.json` for single-page routing.

## Important Files

- `index.html` - SEO tags and app shell
- `src/main.js` - React website and interactions
- `src/menu-data.js` - searchable menu and restaurant data
- `src/styles.css` - premium black/gold styling and animations
- `public/images/` - uploaded restaurant, food, menu, and Google profile images
